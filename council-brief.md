# Council Brief — Taxonomy Harness

## Question

Paul believes the data taxonomy problem in the Agentic AI era can be solved through a reusable harness. Review the project concept and advise:

1. Is this a strong wedge for MarketFitLab?
2. Should it start as workshop, SaaS, or internal harness?
3. What should the MVP include/exclude?
4. What are the biggest risks?
5. What would make customers pay now?

## Context

Agentic AI can connect to Mixpanel MCP and generate insights. But if event taxonomy is messy, AI will confidently produce wrong or unstable analysis. Teams now need AI-readable taxonomy: consistent event names, property definitions, funnel semantics, and data contracts.

## Proposed product

Taxonomy Harness / AI Analytics Readiness Workshop:

- Ingest event schema
- Detect duplicates/ambiguity
- Normalize canonical taxonomy
- Validate with AI analyst question set
- Output taxonomy contract + migration plan + AI-readiness score

## Initial recommendation

Start with a 2–3 hour paid workshop, not SaaS. Use Mixpanel as initial wedge. Productize deliverables as markdown/csv templates and a repeatable validation harness.
